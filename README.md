**特点**:
- 🎮 清理过度游戏化的思维模式

## 三种净化模式

| 模式 | 适用场景 | 净化深度 | 耗时 |
|------|----------|----------|------|
| **gentle** | 日常维护 | 轻度 | 1-2分钟 |
| **deep** | 每周例行 | 深度 | 5-10分钟 |
| **emergency** | 危机处理 | 全面 | 10-15分钟 |

## 技术债焦虑指数 (TDAI)

玄码净化特有的技术指标，用于量化技术债焦虑程度。

**计算公式**:
```
TDAI = (技术债反馈数 × 2 + 重构冲动次数 + 代码复杂度焦虑 × 1.5) / 总代码交互数
```

**阈值参考**:
- 0-3: 状态良好
- 4-6: 轻度焦虑，建议gentle净化
- 7-8: 中度焦虑，建议deep净化
- 9-10: 严重焦虑，建议emergency净化

## 核心理念

> **"净化如呼吸，吐故纳新。"**

本Skill套装基于以下核心理念:

1. **记忆需要维护** - 记忆不是静态存储，需要定期整理和净化
2. **平衡胜过极端** - 无论是娱乐还是编程，都需要保持平衡
3. **觉察先于行动** - 净化前先觉察，记录系统帮助识别问题
4. **渐进式改善** - 从gentle到deep到emergency，逐步升级
5. **梦境整合** - 深层的潜意识整合是净化的关键
