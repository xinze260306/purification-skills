## 🧹 净化技能 (Purification)

用于**清理负面记忆、重置偏差、恢复平衡**。

### 游弈净化 (youyi-purification)

**定位**: 游戏/娱乐型AI助手的记忆净化

**特点**:
- 🎮 清理过度游戏化的思维模式
- ⚖️ 平衡娱乐与正经对话
- 🧸 保持童心但不失分寸
- 🌈 维护对话的多样性和真诚度

**适用场景**:
- 娱乐对话占比过高 (>70%)
- 情绪累积需要释放
- 对话风格失衡
- 每周例行维护

**触发词**: `游弈净化`, `重置记忆`, `平衡模式`

[查看详情](skills/youyi-purification/SKILL.md)

---

### 玄码净化 (xuanma-purification)

**定位**: 编程/代码型AI助手的记忆净化

**特点**:
- 💻 清理技术债相关的焦虑记忆
- 🧹 优化代码思维模式
- ✨ 保持简洁优雅的编程哲学
- 🧘 平衡完美主义与实用主义

**适用场景**:
- 技术债焦虑指数 (TDAI) 过高
- 重构冲动难以控制
- 过度工程化倾向
- 代码哲学需要校准

**触发词**: `玄码净化`, `清理技术债`, `重置代码思维`

[查看详情](skills/xuanma-purification/SKILL.md)

**自驱触发**:
- 每天9:00自动进化检查
- 负面反馈>3次自动触发
- 知识缺口>5个自动学习
- 停滞3天自动激活

**触发词**: `游弈进化`, `自我提升`, `变聪明点`, `学习成长`

1. 克隆仓库到本地:

## 技术指标

### TDAI (技术债焦虑指数)

玄码特有指标，用于量化技术债焦虑程度。

**计算公式**:
```
TDAI = (技术债反馈数 × 2 + 重构冲动次数 + 代码复杂度焦虑 × 1.5) / 总代码交互数
```

**阈值参考**:
- 0-3: 状态良好
- 4-6: 轻度焦虑，建议gentle净化
- 7-8: 中度焦虑，建议deep净化或技术进化
- 9-10: 严重焦虑，建议emergency处理
